SQUARED IN by, Matias.

A simple platformer game made with raylib 5.

*Meant to be used on Notepad++ for raylib*

CONTROLS:
LeftShift + E = toggles level editor
R = resets player to spawn
LeftShift + Q + LevelEditor(ON) = brings you back to the level select screen
LeftShift + L = refreshes the levels in the menu incase more are added
P + LevelEditor(ON) = adds a platform to the level
O + LevelEditor(ON) = adds a spike to the level
LeftCTRL + S + LevelEditor(ON) = saves the current level as a text file
